# Future Claude HTML shells

Drop filled IEP / 504 / MTSS HTML forms here. Link styles with:

```html
<link rel="stylesheet" href="../nocturne/styles.css" />
<link rel="stylesheet" href="../nocturne/print.css" />
```

Use Nocturne classes: `.card`, `.field`, `.input`, `.table`, `.btn`, `.btn-primary`.

Then tell PRISM to register them in Forms Library / Creation Station.
