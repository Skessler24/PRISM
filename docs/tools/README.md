## LRE Calculation Tool for IEPs

Source spreadsheet exported by Samantha Kessler (CCSD / ARR).

- File: [`LRE_Calculation_Tool_for_IEPs.csv`](./LRE_Calculation_Tool_for_IEPs.csv)
- Live UI: React **Quick Tools** → `/tools` (LRE Calculator + Age Calculator)

### Logic mirrored in app

1. Enter weekly minutes each provider pulls the student **outside** general education.
2. Do **not** count push-in services delivered inside gen-ed.
3. School week minutes: default **1,965** (sheet default) or compute hours × 60 × days.
4. LRE % = minutes in GE ÷ total school minutes.
5. Bands: 80–100% Least Restrictive · 40–79% Moderate · 0–39% Most Restrictive.
